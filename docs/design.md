# Age of Crowns — 설계 문서

- 작성일: 2026-07-05
- 설계: 슈퍼천재 (아이디어·밸런스) + 소니 (구조·초안)
- 상태: 슈퍼천재 검토 대기

---

## 1. 게임 소개

나만의 왕국을 짓고, 지키고, 상대 왕국을 함락시키는 **탑다운 2D 픽셀 왕국 샌드박스**.
끝이 없는 세계에서 적 왕국들과 싸우며 칭호를 올린다 — 남작에서 황제까지.

**한 줄 컨셉:** 짓고 → 막고 → 쳐들어가고 → 왕관을 쓴다.

| 항목 | 결정 |
|---|---|
| 장르 | 왕국 건설 + 디펜스 + 정복 샌드박스 |
| 시점 | 탑다운 (위에서 내려다보기) |
| 그래픽 | 2D 픽셀 아트 (고퀄리티) |
| 플랫폼 | PC 브라우저 우선 (모바일은 나중에) |
| 조작 | 왕국 운영 기본 + 유닛 직접 명령 가능 (3단계) |
| 맵 | 매판 랜덤 생성 |
| 엔진 | Phaser 3 + Vite (JavaScript) |
| 배포 | GitHub → Vercel (`buoshllim` 계정, soccer-hub 패턴) |
| 로컬 | `~/Projects/tyground/age-of-crowns/` |

---

## 2. 게임 시작 흐름

1. **타이틀 화면** — 게임 로고 + "게임 플레이 하기" 버튼
2. 버튼 클릭 → **"로딩 중..."** 화면, 5~8초 사이 무작위 시간 (진행바 표시)
3. 로딩 끝 → 랜덤 맵 생성 완료, 게임 시작

---

## 3. 맵과 영토

- 맵 크기: 160×160 타일 (1단계는 100×100으로 시작해도 됨)
- 매판 랜덤 배치: 숲(나무), 돌산(돌), 광맥(철·금·다이아), 강, 각 왕국 위치
- 시작 시 **적 왕국 3개 이상** (1단계에서는 1개로 시작, 3단계에서 3개+)
- **영토**: 각 왕국은 왕성 중심의 영역을 가짐. 자기 영토 안에서만 건설 가능
- **정복**: 적 왕국 함락 시 그 영토가 내 영토가 됨 → 건설 가능 지역 확장

### 함락 판정 — 왕성 (소니 추가 제안 ⚠️)

원래 건물 목록에는 없었지만, "함락"을 판정할 심장이 필요해서 **왕성(Castle)**을 추가 제안.

- 모든 왕국은 시작할 때 왕성 1개를 자동으로 가짐 (건설 불가, 유일)
- **왕성이 파괴되면 그 왕국은 함락** — 남은 건물·유닛도 함께 사라짐
- 슈퍼천재가 다른 함락 기준을 원하면 검토 때 말할 것

## 4. 자원과 경제

### 자원 종류

| 자원 | 어디서 | 쓰임새 | 단계 |
|---|---|---|---|
| 🪵 나무 | 벌목장 (숲 근처) | 건물 건설, 활·창 재료 | 1 |
| 🪨 돌 | 채석장 (돌산 근처) | 건물 건설, 성벽·탑 | 1 |
| 🌾 식량 | 농장 (쌀·밀·당근·감자·순무) | 유닛 소환, 주민 생성 가속 | 1 |
| ⛏ 철 | 광산 | 검·갑옷·석궁 재료 | 2 |
| 🪙 금 | 광산 | 기사 장비·지팡이 재료 | 2 |
| 💎 다이아 | 광산 (희귀) | 마법사 지팡이 재료 | 2 |

### 자원 사슬 (게임의 뼈대)

```
숲 → 벌목장 → 나무 ─┐
돌산 → 채석장 → 돌 ─┼→ 건물 건설
농장 → 농작물 ──────┼→ 주택 (주민 생성 가속) + 유닛 소환 식량
광맥 → 광산 → 철·금·다이아 → 대장간 → 무기(검·활·창·석궁·지팡이) → 훈련소에서 군사 소환
남는 자원 → 창고 보관 → 주민이 필요한 건물로 운반
```

### 주민

- 주택에서 자동 생성 (기본 20초당 1명, 농작물 소비 시 10초당 1명)
- 주택 1채당 주민 최대 3명 (인구 상한 = 주택 수 × 3)
- 주민은 자동으로 일함: 생산 건물 배치, 창고 ↔ 건물 운반
- 플레이어가 주민을 직접 조종하지 않음 (왕국 운영에 집중)

## 5. 건물 (11종 + 왕성)

| 건물 | 비용(초안) | 체력 | 기능 | 단계 |
|---|---|---|---|---|
| 👑 왕성 | 시작 시 자동 | 1000 | 왕국의 심장. 파괴 = 함락 | 1 |
| 🏠 주택 | 나무 30 | 300 | 주민 생성·거주. 농작물 있으면 생성 2배 | 1 |
| 🪓 벌목장 | 나무 20 | 250 | 숲 근처 배치 → 나무 생산 (3초당 1) | 1 |
| ⛏ 채석장 | 나무 30 | 250 | 돌산 근처 배치 → 돌 생산 (3초당 1) | 1 |
| 🌾 농장 | 나무 25 | 200 | 농작물 생산 (3초당 1) | 1 |
| ⚔️ 훈련소 | 나무 50 + 돌 30 | 400 | 자원/무기 소모 → 유닛 소환 | 1 |
| 🧱 성벽 | 돌 10/칸 | 500 | 통행 차단. 적은 부수고 지나가야 함 | 1 |
| ⛏ 광산 | 나무 40 + 돌 40 | 300 | 광맥 위 배치 → 철·금·다이아 생산 | 2 |
| 🔨 대장간 | 나무 50 + 돌 50 | 350 | 광물 → 무기 제작 (검·활·창·석궁·갑옷·지팡이) | 2 |
| 📦 창고 | 나무 40 | 300 | 자원 보관. 주민이 꺼내서 필요한 건물로 운반 | 2 |
| 🗼 탑 | 돌 60 | 450 | 사거리 내 적에게 자동으로 화살 발사 | 2 |
| 🚪 성문 | 나무 20 + 돌 40 | 600 | 아군만 통과. 적은 공격해서 부숴야 함 | 2 |

### 성벽·성문·탑 연결 규칙

- 성문과 탑은 **성벽에 인접한 칸에만** 건설 가능 (서로 연결되어야 함)
- 성벽은 자유롭게 이어 지을 수 있음 (왕국을 빙 둘러싸기 가능)

## 6. 유닛 (6종) — 슈퍼천재 승인 완료 ✅

### 능력치표 v1

| 유닛 | 체력 | 공격력 | 사거리 | 속도 | 비용 | 역할 | 단계 |
|---|---|---|---|---|---|---|---|
| ⚔️ 전사 | 100 | 12 | 근접 | 보통 | 식량 30 | 기본 탱커, 최전방 | 1 |
| 🏹 궁수 | 60 | 10 | 원거리(김) | 보통 | 식량 20 + 나무 20 | 후방 딜러, 유리몸 | 1 |
| 🔱 창병 | 80 | 10 | 근접(살짝 김) | 느림 | 식량 25 + 나무 10 | 대기사 방패부대 | 1 |
| 🎯 석궁병 | 65 | 16 | 원거리(중간) | 느림 | 식량 25 + 철 15 | 갑옷 뚫는 저격수 (공속 느림) | 2 |
| 🐴 기사 | 180 | 18 | 근접 | 빠름 | 식량 50 + 철 40 + 금 10 | 돌격대장, 원거리 사냥꾼 | 2 |
| 🔮 마법사 | 50 | 20 (범위) | 원거리 | 느림 | 식량 40 + 금 30 + 다이아 5 | 광역 폭발 + 건물 킬러, 초유리몸 | 2 |

*2단계부터 유닛 소환에 대장간 무기 필요 (검·활·창·석궁·지팡이). 위 표의 광물 비용이 무기 재료.*

### 상성표 v1 — 상성 맞으면 데미지 2배 (감소 상성 없음)

| 공격 유닛 | 2배로 잡는 상대 |
|---|---|
| ⚔️ 전사 | 궁수, 석궁병 |
| 🏹 궁수 | 창병 |
| 🔱 창병 | 전사, 기사 |
| 🎯 석궁병 | 기사 |
| 🐴 기사 | 궁수, 마법사 |
| 🔮 마법사 | (상성 대신) 범위 폭발 + 건물에 2배 |

```
1단계 삼각형: 전사 → 궁수 → 창병 → 전사
2단계: 창병·석궁병 → 기사 → 궁수·마법사
```

모든 유닛에 천적이 있음 → 무적 조합 없음.

### 레벨업

- 적 처치 시 경험치 → 레벨 1~5
- 레벨업마다 체력 +20%, 공격력 +10%
- 죽으면 레벨 소멸 → 고참병을 지키는 재미

## 7. 전투와 조작

- **기본 (1단계)**: 유닛은 자동 전투. 시야 내 적을 우선순위(가까운 적 먼저)로 공격
- **공격 깃발 (1단계)**: 맵에 깃발을 꽂으면 군사들이 그 지점으로 진군
- **직접 명령 (3단계)**: 드래그로 유닛 선택 → 우클릭 이동/공격 명령 (RTS 조작)
- 성문은 아군만 통과. 적 유닛은 성벽·성문을 공격해서 부수고 진입

---

## 8. 적 AI 왕국

- 적 왕국도 플레이어와 **같은 규칙**으로 움직임: 자원 채집 → 건설 → 군사 생산 → 공격/방어
- AI 행동 상태: `성장 → 방어 → 공격` 순환 (자원·군사 상황에 따라 전환)
- **적끼리도 전쟁함**: 주기적으로 공격 대상을 정함 (플레이어 또는 다른 적 왕국)
- **리스폰**: 왕국이 함락되어 사라지면 맵 가장자리 빈 영역에 새 왕국 등장 → 세계에 항상 적 왕국 3개 이상 유지
- 새 왕국은 기존과 같은 규칙으로 시작 (밸런스는 플레이테스트 후 조정)

---

## 9. 진행 동기 — 칭호 시스템

적 왕국을 함락시킬 때마다:

1. **영토 확장** — 함락한 땅이 내 영토가 됨
2. **칭호 상승** — 함락 횟수 기준

| 함락 횟수 | 칭호 |
|---|---|
| 0 | 🏡 남작 |
| 1 | 🎖 자작 |
| 2 | 🏅 백작 |
| 3 | 🎗 후작 |
| 4 | 🏰 공작 |
| 6 | 👑 왕 |
| 8 | 🌟 황제 |

칭호는 화면 상단에 항상 표시. 황제 이후에도 게임은 계속 (샌드박스).

## 10. 개발 단계 (수직 슬라이스)

### 1단계 — 왕국의 탄생 ("짓고 → 막고 → 함락"이 한 판에 다 되는 최소 게임)
- 타이틀 + 로딩(5~8초 랜덤), 랜덤 맵, 카메라 스크롤
- 자원 3종(나무·돌·식량), 건물: 왕성·주택·벌목장·채석장·농장·훈련소·성벽
- 주민 자동 생성·자동 일하기
- 유닛 3종(전사·궁수·창병) + 상성 삼각형 + 공격 깃발
- 적 왕국 1개 (건설·생산·공격하는 AI), 왕성 파괴 = 함락 + 칭호 1단계 상승

### 2단계 — 경제의 완성
- 광산(철·금·다이아), 대장간 무기 사슬, 창고 물류
- 유닛 6종 전부 + 레벨업, 성문·탑 + 성벽 연결 규칙

### 3단계 — 살아있는 세계
- 적 왕국 3개+, 적끼리 전쟁, 함락 → 영토 확장, 리스폰
- 칭호 시스템 전체, 유닛 드래그 선택·직접 명령

### 4단계 — 반짝반짝
- 슈퍼천재 제작 에셋·음악·효과음 적용
- 저장/불러오기 (localStorage), 밸런스 조정

각 단계 완료 기준: **그 단계만으로 플레이 가능한 게임일 것.**

### 5단계 이후 후보 (백로그 — 확정 아님, 잊지 않기 위한 목록)

- **모바일·태블릿 터치 대응** — 플랫폼 결정 때 "PC 우선, 모바일은 나중에"로 미룬 것. 터치로 건설·유닛 선택·카메라까지 되려면 UI 재설계 필요. 4단계까지 끝나고 재미가 검증되면 착수 여부 결정
- 상성표 확장 검토 — 2단계 유닛 6종이 다 들어온 뒤 플레이해보고 필요하면
- itch.io 배포 — Vercel 외 추가 배포처 후보

## 11. 에셋 계획 (외부 제작 — 슈퍼천재 담당)

개발 중(1~3단계)에는 **무료 픽셀 팩**(Tiny Swords 등 탑다운 중세 팩)으로 진행하고, 4단계에서 슈퍼천재 제작 에셋으로 교체. 팩에 없는 건물은 색깔 사각형으로 임시 처리. 타일 크기는 처음부터 32×32로 통일 (교체 시 그림만 갈아끼우면 되게).

- **유닛 스프라이트**: 6종 × (걷기·공격·죽기) 애니메이션. 한 벌만 그리고 팀 색만 바꿔 재사용
- **건물 스프라이트**: 12종 (왕성 포함) + 부서진 모습
- **타일**: 잔디·숲·돌산·광맥·강·길
- **UI**: 자원 아이콘 6종, 칭호 아이콘 7종, 버튼
- **음악**: 타이틀 1곡, 평화 1곡, 전투 1곡
- **효과음**: 칼 부딪힘, 화살, 폭발(마법), 건설, 건물 파괴, 함락 팡파레

정확한 규격(픽셀 크기, 프레임 수)은 1단계 구현 후 소니가 목록으로 정리해서 전달.

## 12. 기술 메모 (구현 담당 Sonnet 참고)

- Phaser 3 + Vite, `pixelArt: true`, 씬 구성: `TitleScene → LoadingScene → GameScene`
- 배포: GitHub push까지만 에이전트 담당, Vercel 연동은 아빠가 직접
- 모바일 대응은 범위 밖 (PC 마우스+키보드 전제)
- 밸런스 숫자(생산 속도, 비용, 능력치)는 전부 **데이터 파일로 분리** — 슈퍼천재가 직접 수정하며 밸런싱하는 구조 (Click Football Tycoon 방식)
- 이 문서의 숫자는 초안. 플레이테스트로 조정하는 것이 전제
